** 使用方法
転送用ファイル作成

$ ./make_randomfile.py {作成したいファイル数}


ファイルのハッシュ取得

$ ./getmd5.py


reciever起動

$ ./recv.py local


sender起動

$ ./send.py 


転送したファイルのハッシュ確認

$ ./checkfile.py
